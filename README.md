# cacei_important_doc
## This repo contains important documentation about the creation of the DB.
Entidades CACEI file
* Contains analisis about main tables, pivot tables and system catalog tables, PK's, Fk's, Constrains, etc.

main_tables file
* Contains the SQL statements to create the main tables of the model, PK's, FK's and constraint.

pivot_tables file
* Contains the SQL statements to create the pivot tables of the model, PK's, FK's and constraint.

sysyem_catalog_table file
* Contains the SQL statements to create the catalog tables of the model, PK's, FK's and constraint.

triggers file
* Contains the SQL statements to create some triggers to execute some procedures to validate and make changes in some tables.

ERR_CACEIV3 file
* This is de Entity Relationship Diagram
